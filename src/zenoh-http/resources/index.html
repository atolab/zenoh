Welcome to the ZENOH REST API

Usage:

GET /a/b/c
get value for resource '/a/b/c'
GET /a/b/*
get all values for resources matching selection '/a/b/*' ('/a/b/c', '/a/b/d', '/a/b/de', ...)
GET /a/b/**
get all values for resources matching selection '/a/b/**' ('/a/b', '/a/b/c', '/a/b/c/d', ...)
GET /@/**
get all values from Admin Space
GET /a/b/c?<predicate>
get value for resource '/a/b/c' with predicate '<predicate>'
PUT /a/b/c "HELLO"
put value 'HELLO' in resource '/a/b/c'
